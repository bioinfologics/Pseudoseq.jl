Pseudoseq

Description

The Pseudoseq package allows you to simulate DNA sequencing experiments. DNA sequencing experiments are modelled conceptually as a sampling process.

Install

You can install Pseudoseq from the julia REPL:

using Pkg
add("https://github.com/bioinfologics/Pseudoseq.jl.git")

Testing

Pseudoseq is tested against Julia 1.X on Linux, OS X, and Windows.
